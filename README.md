This repository hosts our **D**enoisi**n**g **A**ttention (DnA) project code and experiments.


## Repository Structure
- `timesformer/`: DnA-modified TimeSformer code for video action recognition experiments.
- `viscop/`: DnA-modified VisCoP code for visual probing and domain adaptation experiments.
- `vit/`: Vision Transformer (ViT) training code and DnA modules for controlled studies.
